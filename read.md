## Run locally

Open `login.html` in Chrome or Microsoft Edge. Click **Select users.csv** and choose this project's `users.csv` file. That CSV is the only credential store: Sign up appends the new account directly to it, and Login validates against the same selected file. No Node.js server or database is used.
